<h1 align="center">District-wise IPC Crimes in India (2017 onwards)</h1>
<p align="center"><b>Exploratory Data Analysis of Indian Penal Code Crimes at District Level</b></p>

<hr>

<h2>📌 Project Overview</h2>
<p>
This project presents a comprehensive data analysis of district-wise IPC crimes in India from 2017 onwards. 
Using an official multi-year dataset, the study explores offence-wise crime patterns across districts and states.
</p>

<ul>
  <li>Temporal trends</li>
  <li>Geographic hotspots</li>
  <li>Variations across crime categories</li>
</ul>

<hr>

<h2>📊 Dataset Description</h2>
<p>
<b>Source:</b><br>
<a href="https://indiadataportal.com/p/crime-statistics/r/ncrb-cii_ipc_crimes-dt-yr-aaa" target="_blank">
India Data Portal - IPC Crimes Dataset
</a>
</p>

<p>
The dataset contains detailed offence-wise crime counts at the district level, covering multiple years 
and enabling both spatial and temporal analysis.
</p>

<hr>

<h2>🎯 Objectives / Problem Statements</h2>

<h3>1. Temporal Trends</h3>
<ul>
  <li>Analyze how major IPC crime categories have evolved from 2017 onwards</li>
  <li>Study trends at national, state, and district levels</li>
</ul>

<h3>2. Geographic Hotspots</h3>
<ul>
  <li>Identify districts and states with consistently high crime volumes</li>
  <li>Examine hotspots for offences such as murder, rape, and theft</li>
</ul>

<h3>3. Crimes Against Women</h3>
<ul>
  <li>Analyze offences like rape, cruelty by husband or relatives, and sexual harassment</li>
  <li>Study district-wise and year-wise variations</li>
</ul>

<h3>4. Property and Economic Crimes</h3>
<ul>
  <li>Examine spatial and temporal patterns of theft, robbery, and dacoity</li>
  <li>Analyze trends in economic offences such as bank and ATM fraud</li>
</ul>

<h3>5. District Crime Profiles</h3>
<ul>
  <li>Cluster districts based on crime composition</li>
  <li>Identify profiles such as:</li>
  <ul>
    <li>High violent crime districts</li>
    <li>High property crime districts</li>
    <li>Mixed-pattern districts</li>
  </ul>
</ul>
